Norgescup 3 og Østlandscup

Start: 2020-09-26 04:00 | Slutt: 2020-09-27 05:59

Lørdag 26. september arrangeres Norges Cup (NC) 3 og Østlandscup. 

Vi vil gjerne ha med de som er interessert i å prøve seg enten i kamp- eller poomsekonkurranse, eller bare har lyst til å være med å se og heie. Prat med instruktørene hvis du ønsker å delta. Anders tar seg av påmelding, og klubben dekker deltakeravgift. Det er foreløpig ikke bestemt hvor det skal avholdes. Det kommer oppdatert informasjon her, så fort det blir tilgjengelig. Se ellers kampsportforbunds hjemmesider  for oppdatert og utfyllende informasjon.
