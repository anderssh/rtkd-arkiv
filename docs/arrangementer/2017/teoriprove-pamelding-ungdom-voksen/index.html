Arrangement

Teoriprøve / Påmelding ungdom og voksen

Start: 2017-12-14 18:00 | Slutt: 2017-12-14 18:10

Påmelding.

14.12.2017 Påmelding og teoriprøve for medlemmer på Ungdom og voksen. Påmelding og betaling foregår denne dato ved oppmøte i Ringerikshallen i kampsport rommet nede kl. 1800. Skal du gradere, så setter du av dato og klokkeslett! Har du noe annet viktig som foregår denne datoen (jobb som ikke kan byttes, viktig arrangement og lignende), så må du ta kontakt med Geir for en avtale, så finner vi en løsning..

Betaling.

Gradering voksen gult til og med blått med rød stripe er kr 400,- og inkluderer nytt belte og diplom Gradering voksen rødt 1,2 og 3 striper koster 450,- og inkluderer, treplater, stripe og diplom.  

Treplater

1 sort stripe = 1 planke. Skal knekkes med yeop chagi. 2 sorte striper = 2 planker. Skal knekkes med slag og så følges opp med yeop chagi. 3 sorte striper = 3 planker. Skal knekkes med slag, valgfritt spark og avsluttes med hoppspark. (yeop chagi)

Teoriprøve.

Denne skal avlegges uten hjelpemidler og for gult belte til og med blått med rød stripe så teller denne som en del av totalvurderingen av din gradering. for rødt belte (4.Gup) og høyere må prøven besvares med 70 % riktige eller høyere. Anbefaler alle å åpne teoriboken og lese seg opp. Tips er å gå inn på siden vår som heter pensum og laste ned heftet som ligger der.

Gradering.

Dette foregår 19.12.2017 og oppmøte er i kampsport rommet nede kl 1700 og vi skal være ferdige til 2100.  

Spørsmål ?

Noe som er uklart, så ta kontakt med Geir.
