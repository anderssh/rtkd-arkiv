Arrangement

Norgescup 2 og Østlandscup

Start: 2019-05-04 08:00 | Slutt: 2019-05-04 18:00

Lørdag 4. mai arrangeres Norges Cup (NC) 2 og Østlandscup.

Etter stor suksess med NC 1, satser vi på minst like godt engasjement igjen på NC 2 på Bygdøhus i 4. mai Vi vil gjerne ha med de som er interessert i å prøve seg enten i kamp- eller poomsekonkurranse, eller bare har lyst til å være med å se og heie. Prat med instruktørene hvis du ønsker å delta. Anders tar seg av påmelding, og klubben dekker deltakeravgift. Facebook-arrangementet: https://www.facebook.com/events/268802104017148/ Se ellers Norges kampsportforbunds hjemmesider for informasjon om dette, og senere stevner.
