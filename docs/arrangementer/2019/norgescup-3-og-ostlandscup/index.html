Arrangement

Norgescup 3 og Østlandscup

Start: 2019-09-28 08:00 | Slutt: 2019-09-28 18:00

Lørdag 28. september arrangeres Norges Cup (NC) 3 og Østlandscup.

Det er ennå ikke bestemt hvor det skal avholdes, men så fort det blir avklart legges det ut her. Det kommer til å være folk fra klubben med. Vi vil gjerne ha med de som er interessert i å prøve seg enten i kamp- eller poomsekonkurranse, eller bare har lyst til å være med å se og heie. Mer informasjon kommer så fort det er tilgjengelig, og se ellers Norges kampsportforbunds hjemmesider for informasjon om dette, og senere stevner.
