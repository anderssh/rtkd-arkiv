Arrangement

Ringerike TKDs tradisjonelle julecamp med gradering

Start: 2019-12-14 12:00 | Slutt: 2019-12-15 16:00 | Sted: Ringerikshallen, Tyrimyrveien, Hønefoss | Arrangør: Ringerike Taekwondo-klubb (www.arkiv.ringeriketkd.no)

Klubben inviterer i år igjen til julecamp for barn- og tigergruppen 14.-15. desember. Det blir trening, gradering, filmvisning, overnatting (for de som vil) og annen julehygge!

  Under finner du invitasjon og program, som kan lastes ned, og påmelding gjøres gjennom påmeldingsskjemaet som lenkes til nederst på siden. Påmeldingsfrist 4. desember. Velkommen til årets begivenhet for barn og tigern! https://arkiv.ringeriketkd.no/pamelding-julecamp-2019/
