Arrangement

Påmelding til og teoriprøve for gradering Ungdom og voksne.

Start: 2018-06-12 19:30 | Slutt: 2018-06-12 20:00

For de som ønsker å gradere på ungdom og voksen partiet så er det påmelding og teoriprøve tirsdag 12.juni klokken 1930. Graderingen koster kroner 400 for beltegrad hvitt tilogmed blått med stripe. og kr 450 for rødt belte og høyere. Dette betales samtidig som påmeling (utlevering av teoriprøve) Det kan betales med kontant, kort og vips. Graderingen vil bli gjennomført lørdag 23.juni klokken 1200-1700. Om vi holder på helt til klokken 1700 er avhenig av hvor mange som skal gradere. Dette har vi ikke svar på før det nærmer seg. Fysisk test avholdes som en del av graderingen. Planker skal knekkes i henholdt til klubbens graderingspensum slik:
  1. sort stripe. 1.planke som skal knekkes med yeopchagi.
  2. sorte striper. 2.plater som skal knekkes med monton jirugi og yeopchagi.
  3. sorte striper. 3 plater som skal knekkes med monton jirugi, yeopchagi og hoppspark med yeopchagi.
