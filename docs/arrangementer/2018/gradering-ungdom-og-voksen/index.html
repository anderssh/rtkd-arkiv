Arrangement

Gradering Ungdom og voksen

Start: 2018-06-23 12:00 | Slutt: 2018-06-23 17:00 | Arrangør: Ringerike Taekwondo-klubb (www.arkiv.ringeriketkd.no)

Graderingen vil bli gjennomført lørdag 23.juni klokken 1200-1700. Om vi holder på helt til klokken 1700 er avhenig av hvor mange som skal gradere. Dette har vi ikke svar på før det nærmer seg. Fysisk test avholdes som en del av graderingen. Planker skal knekkes i henholdt til klubbens graderingspensum slik:
  1. sort stripe. 1.planke som skal knekkes med yeopchagi.
  2. sorte striper. 2.plater som skal knekkes med monton jirugi og yeopchagi.
  3. sorte striper. 3 plater som skal knekkes med monton jirugi, yeopchagi og hoppspark med yeopchagi.
HUSK PÅMELDING DEN 12.juni.
