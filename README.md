# Ringerike Taekwondo Klubb Arkiv

Dette repoet inneholder en statisk, lesbar arkivversjon av den tidligere WordPress-siden.

## Struktur

- `docs/` inneholder selve nettstedet og er ment som GitHub Pages-kilde
- `docs/index.html` er forsiden til arkivet
- `docs/CNAME` peker til `arkiv.ringeriketkd.no`
- `docs/.nojekyll` sørger for at GitHub Pages publiserer filene som de er

## Åpne arkivet

- Lokalt: åpne `docs/index.html`
- Publisert: bruk `https://arkiv.ringeriketkd.no/`

## Viktige filer

- [Forside](docs/index.html)
- [Historie](docs/historie-2/index.html)

- [Innlegg](docs/innlegg/index.html)

Anbefalt oppsett er GitHub Pages fra `main`-grenen og mappen `docs/`.

Se `DEPLOY.md` for kort oppskrift.
